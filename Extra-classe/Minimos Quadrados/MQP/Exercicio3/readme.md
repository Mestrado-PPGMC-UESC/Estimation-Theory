# Exercício 6 — Mínimos Quadrados Ponderados (Modelo Quadrático)

Durante testes de frenagem de um veículo autônomo, foi registrada a distância percorrida pelo veículo após o acionamento do sistema de frenagem.

Sabe-se que, devido a diferentes condições dos sensores, cada medição apresenta um nível de confiança distinto.

Os dados experimentais são apresentados abaixo:

Tempo (s)	Distância (m)	Confiança
0	102.0	5
1	93.8	5
2	76.5	4
3	51.4	3
4	18.2	2
5	-24.6	1

Deseja-se modelar a posição do veículo por uma função quadrática da forma:

[
s(t)=a+bt+ct^2
]

Utilizando o Método dos Mínimos Quadrados Ponderados, determine os coeficientes:

[
a,\ b,\ c
]

que melhor ajustam os dados experimentais.
