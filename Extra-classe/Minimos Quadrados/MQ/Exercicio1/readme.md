# Exercício 1 — Método dos Mínimos Quadrados

Utilizando o **Método dos Mínimos Quadrados**, determine a função linear do tipo:

[
f(x)=a+bx
]

que melhor aproxima os pontos experimentais apresentados na tabela abaixo.

| x    | 0 | 1 | 2 | 3 |
| ---- | - | - | - | - |
| f(x) | 1 | 2 | 4 | 8 |

O objetivo é encontrar os coeficientes (a) e (b) que minimizam o erro quadrático entre os valores observados e os valores estimados pelo modelo linear.
